# 这是什么
MultiDrive是一款云盘管理程序，支持像oneindex那样列出云盘文件目录，提供基本的网盘管理功能。  
很多功能仍在开发...

# 功能
- [x] 自动刷新token
- [x] 文件直链下载
- [x] 渲染readme.md文件
- [x] 基本图片、文本、视频预览
- [x] Redis缓存，加快列表速度
- [x] 分享指定目录
- [x] 将部分格式的office文件转换为PDF

# 支持网盘
- [x] OneDrive
- [x] AliyunDrive
- [x] 本地存储
- [ ] GoogleDrive
- [ ] FTP
- [ ] 七牛 / 又拍

# 安装方法
```shell
pip install -r requirements.txt
```
